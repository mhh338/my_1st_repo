# CREATING REPO ON GITHUB
echo "# my_1st_repo" >> README.md 
('echo' command writes contents enclosed within " " into 'README.md' file,
 '>>' used to overwrite the 'README.md' file, '#' is used for content to be appear as heading in the file.)

git init 
(Initializes the repo.)

git add .
(Adds all the files to the stage.)

git config --global user.name "--NAME" 
(Configures your name. Required only once.)

git config --global user.email "--EMAIL"     
(Configures your email. Required only once.) 

git add README.md
(Adds the 'README.md' to the stage which is then to be pushed.)

git commit -m "first commit"
(Adds a 'commit' message to the staged file.)

git branch -M main
(Creates a branch named 'main'.)

git remote add origin https://github.com/mhh338/my_1st_repo.git
(Connects the local repo on the computer to github repo.)

git push -u origin main
(Puhes the staged file to the repo of origin given in previous step. 
Here '-u' is used so that after wards user can only write 'git push' and it pushes to the 'main' branch.)

git clone <repo-URL>
(Clone a repository)

git cone -b <branch-name> <repo-URL>
(Clone a particular branch from a repository)
